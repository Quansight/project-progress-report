Quansight Progress Report

7/3/2020

Project Overview

Quansight proposes to build a flexible MES solution based on a mixed open source/commercial tech stack. The commercial component would be the KEPServerEX connectivity platform from KEPWare. This platform has confirmed support of 9 of the 16 based on exact machine model numbers as well as probable support of the other 7 machine types based on supported protocols and/or support for a family of machines. It has a REST API that can be utilized to control and interact with the platform. Using this supported commercial product will reduce the risk in the most uncertain area of the project which is the interface with the factory machines.
